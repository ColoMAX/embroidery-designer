# Embroidery Design Tool

A simple web app to help create embroidery designs from images. Upload an image, resize it, reduce the color palette, and download the result.

To use it in your browser now, visit [https://colomax.github.io/embroidery-designer/](https://colomax.github.io/embroidery-designer/)

## Features

- Upload images
- Resize to custom dimensions
- Choose from predefined color palettes (embroidery, basic, grayscale) or specify number of colors
- Preview the processed image
- Download as PNG
- Print functionality

## How to Use

1. Choose an image file
2. Set the desired width and height
3. Select color option: predefined palette or number of colors
4. Click "Process Image" to see the preview
5. Download the PNG or print the design

## Technologies

- HTML5 Canvas for image processing
- RgbQuant.js for color quantization
- Pure JavaScript, no server required
